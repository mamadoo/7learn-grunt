# 7learn-grunt
Files and folder structure for Grunt 7learn course

## Get started
First open cmd and run this command:
~~~sh
npm install
~~~
After install all developer dependencies, you can run grunt with this command:
~~~sh
npm start
~~~
or:
~~~sh
grunt
~~~
